##Lee Eden Wedding DJ Service

This is the personal website of **Lee Eden Wedding DJ service.**

It is based on the { personal } theme for jekyll which is itself based on the grayscale theme

config.yml
==========

this is the main configuration file. Most of the key decisions can be modified in this file.
Follow the format.

use # to comment out lines (so that the computer ignores these lines)

Editing text for the different pages
====================================

The text body for each page is in the main directory and has the title of each page

- index.html is for the front section
- home.html is for the home section
- about.html is for the about section
- services is for the services and prices section
- gallery is for the gallery section
- contacts is for the contact section
